# Bug Bounty Bot

Ferramenta automatizada para reconhecimento em programas de Bug Bounty, realizando descoberta de ativos, scan de portas, identificação de tecnologias e detecção de vulnerabilidades.

## 🚀 Características

- **Descoberta Passiva de Subdomínios**: Integração com múltiplas fontes (crt.sh, HackerTarget, AlienVault, Wayback Machine, Common Crawl, RapidDNS)
- **Descoberta Ativa**: Brute force de subdomínios com wordlists customizáveis
- **Verificação de Hosts Ativos**: Probe HTTP/HTTPS para identificar serviços em execução
- **Scanner de Portas**: Detecção de portas abertas com suporte a nmap
- **Descoberta de Endpoints**: Identificação de URLs e paths sensíveis
- **Fingerprint de Tecnologias**: Identificação automática de stacks tecnológicas
- **Scanner de Vulnerabilidades**: Integração com Nuclei e verificações customizadas
- **Correlação de Dados**: Análise e priorização de riscos
- **Múltiplos Formatos de Relatório**: HTML, JSON e Markdown
